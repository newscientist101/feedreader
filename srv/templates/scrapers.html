            Configuration Help
            

                
HTML Scraper (CSS Selectors)

                
For scraping HTML pages using CSS selectors:

                
    
                    
  • itemSelector - CSS selector for each item container (required)
    • 
                    
  • titleSelector - Selector for the title element (text content is used)
    • 
                    
  • urlSelector - Selector for the link element
    • 
                    
  • urlAttr - Attribute to extract URL from (default: href)
    • 
                    
  • summarySelector - Selector for description text (optional)
    • 
                    
  • authorSelector - Selector for author name (optional)
    • 
                    
  • imageSelector - Selector for image element (optional)
    • 
                    
  • imageAttr - Attribute to extract image URL from (default: src)
    • 
                    
  • dateSelector - Selector for date element (optional)
    • 
                    
  • dateAttr - Attribute to extract date from (if empty, uses text content)
    • 
                    
  • baseUrl - Base URL for resolving relative links
    • 
                


                
JSON Scraper

                
For scraping JSON APIs. Set "type": "json" to enable:

                
    
                    
  • type - Must be "json"
    • 
                    
  • itemsPath - Dot-separated path to the items array (e.g. "data.items")
    • 
                    
  • titlePath - Dot-separated path to title within each item
    • 
                    
  • urlPath - Path to URL
    • 
                    
  • summaryPath - Path to summary/description
    • 
                    
  • contentPath - Path to full content
    • 
                    
  • authorPath - Path to author
    • 
                    
  • imagePath - Path to image URL
    • 
                    
  • datePath - Path to date/timestamp
    • 
                    
  • guidPath - Path to unique ID
    • 
                    
  • baseUrl - Base URL for resolving relative links
    • 
                    
  • consolidateDuplicates - Merge consecutive items with same title, deduping sentences (boolean)
    •
