HTML Scraper (CSS Selectors)

                            

                                
type
Must be "html" (required)

                                
itemSelector
CSS selector for each item container (required)

                                
titleSelector
Selector for title element (text content)

                                
urlSelector
Selector for link element

                                
urlAttr
Attribute for URL (default: href)

                                
summarySelector
Selector for description text

                                
authorSelector
Selector for author name

                                
imageSelector
Selector for image element

                                
imageAttr
Attribute for image URL (default: src)

                                
dateSelector
Selector for date element

                                
dateAttr
Attribute for date (text content if empty)

                                
baseUrl
Base URL for relative links

                            

                        

                        

                            
JSON API Scraper

itemsPath

titlePath

urlPath
Dot path to URL

                                
summaryPath
Dot path to summary/description

                                
contentPath
Dot path to full content

                                
authorPath
Dot path to author

                                
imagePath
Dot path to image URL

                                
datePath
Dot path to date/timestamp

                                
guidPath
Dot path to unique ID

                                
baseUrl
Base URL for relative links

                                
consolidateDuplicates
Merge consecutive same-title items (boolean)
