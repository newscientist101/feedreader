About Scraper Modules

Scraper modules let you extract content from websites that don't have RSS feeds. Each module is a JSON configuration with regex patterns to extract article data.

Example Configuration 
{
  "itemPattern": "<article[^>]*>.*?</article>",
  "titlePattern": "<h2[^>]*>([^<]+)</h2>",
  "urlPattern": "href=\"([^\"]+)\"",
  "summaryPattern": "<p class=\"summary\">([^<]+)</p>",
  "datePattern": "<time[^>]*>([^<]+)</time>",
  "baseUrl": "https://example.com"
}
